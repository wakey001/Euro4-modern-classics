An Introduction to the Honda Civic Type R 2001-2006
The 2001-2006 Honda Civic Type R .In 2001, Honda introduced the next generation of the Civic Type R as a unique 3-door hatchback to the UK market, which was manufactured in Swindon, England.

Civic Type R featured a 200 PS (147 kW; 197 hp) 2.0-liter i-VTEC engine
